Terrain Generation (Natural Phase)

Noise layers determine terrain type. The autotile renderer picks sheets based on each tile + its neighbors:

  1. Elevation noise → deep water / shallow water / land
  2. Moisture noise → sand (dry) / dirt (normal) / grass (wet)
  3. Variety noise → which dirt color (#1 light vs #2 warm)
  4. For each border between terrain types, select the matching dedicated pair sheet (or fall back to alpha overlay)
  5. Props scattered on top for visual detail
